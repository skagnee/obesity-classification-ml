# 🧠 Obesity Risk Classification using Machine Learning

## 📌 Project Overview
This project focuses on analyzing lifestyle and health-related factors to predict obesity levels using multiple machine learning algorithms. A comparative study is performed to evaluate different classification models and identify the most effective one.

---

## 🎯 Objective
- Perform data preprocessing and cleaning  
- Analyze patterns using descriptive statistics and visualization  
- Apply multiple classification algorithms  
- Compare model performance using evaluation metrics  
- Identify important features influencing obesity  

---

## 📊 Dataset
- Dataset: Obesity Dataset (Raw and Synthetic)
- Contains attributes related to:
  - Eating habits
  - Physical activity
  - Lifestyle patterns
  - Demographic information  

---

## ⚙️ Workflow

### 🔹 1. Data Preprocessing
- Handling missing values  
- Encoding categorical variables  
- Feature scaling  

### 🔹 2. Exploratory Data Analysis (EDA)
- Distribution plots  
- Correlation analysis  
- Visualization of key variables  

### 🔹 3. Outlier Detection
- IQR Method  
- Z-Score  
- DBSCAN (Multivariate outlier detection)  

### 🔹 4. Feature Engineering
- Selection of relevant features  
- Transformation where necessary  

---

## 🤖 Machine Learning Models Used
- Logistic Regression  
- Decision Tree  
- Random Forest  

---

## 📈 Model Evaluation
Models are compared using:
- Accuracy  
- Precision  
- Recall  
- F1-Score  

---

## 🏆 Results
- Random Forest performed the best among all models  
- Provided higher accuracy and better generalization  

---

## 📌 Key Insights
- Lifestyle factors significantly influence obesity levels  
- Feature importance analysis highlights critical predictors  
- Proper preprocessing improves model performance
